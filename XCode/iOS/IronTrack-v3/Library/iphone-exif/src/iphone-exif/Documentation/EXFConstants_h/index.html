EXFDataType

The possible types that an EXIF tag data can be specified as. 

enum EXFDataType { 
    FMT_BYTE = 1, 
    FMT_STRING = 2, 
    FMT_USHORT = 3, 
    FMT_ULONG = 4, 
    FMT_URATIONAL = 5, 
    FMT_SBYTE = 6, 
    FMT_UNDEFINED = 7, 
    FMT_SSHORT = 8, 
    FMT_SLONG = 9, 
    FMT_SRATIONAL =10, 
    FMT_SINGLE =11, 
    FMT_DOUBLE =12 
};
Discussion

These are the only legal types to be used in the EXFTagDefinition to determine the type of data to be read/written and the number of bytes that each data type will then occupy.
